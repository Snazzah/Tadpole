Tadpole

A Discord bot that connects to multiple channels.

Download this project as a .zip file Download this project as a tar.gz file

Commands

host

#tadpole host|make|create

Creates a connection.

join

#tadpole join|connect (tadpole id)

Joins a connection.

end

#tadpole end|leave|disconnect

Disconnects the channel from the current connection. The connection ends if there were only 2 channels connected.

cinfo

#tadpole cinfo|connectioninfo

Displays connection info.
